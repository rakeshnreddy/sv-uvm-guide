# Findings Directory

Store QA defects and discrepancies here for the implementation agent.

## Naming Convention
`YYYY-MM-DD_<feature-id>_<short-title>.md`

Example:
`2026-02-27_W5-IUVM2-SPLIT_tlm-port-wiring-mismatch.md`

## Required Contents
- Severity (`P0`..`P3`)
- Affected files/modules
- Reproduction steps
- Expected behavior
- Actual behavior
- Suggested fix direction
- Validation steps to close

Use `FINDING_TEMPLATE.md` as the source format.
